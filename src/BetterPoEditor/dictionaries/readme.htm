BetterPoEditor uses the NetSpell component for the spell check operations.

If the dictionary for your language is not included in the standard distribution of BetterPoEditor you can create it in a very simple manner, following there steps:

  1. Browse to the the OpenOffice Extensions repository
  2. Locate and download the dictionary pack for the language you need
  3. The downloaded file has an .oxt extension, but it is a zip archive. Extract the .dic and the .aff files that it contains with your favorite archive manager (7-zip can do that for example).
  4. Launch the DictionaryBuild application that comes with BetterPoEditor (DictionaryBuild is part of the NetSpell project).
  5. Create a new empty dictionary (File → New)
  6. Import the .aff file with Dictionary → Add OpenOffice Affix Data.
  7. Import the .dic file with Dictionary → Add OpenOffice Word List.
  8. Save (File → Save) the dictionary in the dictionaries folder present in the installation folder of BetterPoEditor (that is this folder).
  9. The file name must be the one signaled by BetterPoEditor.
